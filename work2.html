Contactless Payments, Chalo

Chalo Card is tap-to-pay contactless Chalo Card. UI/UX designs for Desktop and Android webviews.


About The Project

A project developed as a Software Developer under Chalo. The project was to develop a User Interface for onboarding customers to Chalo Card (in partnership with Yes bank) for KYC and creating offline wallets. A dedicated online web portal for user account management. At this project, I acted as a Front End/Angular Developer. The challenge was to design lightweight interactive SPA to be used as a website and also as an Webview inside Android App. The time to complete KYC was tremendously brought down to less than 2 seconds. As the full development process was made online using this web app, the management and financial transparency of all user info were made easy.

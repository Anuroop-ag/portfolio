Speak - Audio/Text

A one to one audio/text chat web application using Vue.

About The Project

A project developed as an Independent Firebase Developer . This project is a methods to send audio files via chat. The main aim of this project was to build a Graphical User Interface for sending audio chat. The main challenge was conversion and storing of audio files.
